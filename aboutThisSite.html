Acerca de este sitio

Este sitio nace como un proyecto del curso de Desarrollo de Aplicaciones Web de la carrera de Ciencias de Computación e Informática de la UCR. En este, se pidió que se hiciera un sitio web personal con una estructura específica.

Este sitio es hosteado en un servidor perteneciente a la Escuela de Ciencias de la Computación e Informática, llamado 'titanic'.

El sitio pertende ser principalmente amigable a dispositivos móviles, y al mismo tiempo ser compatible con todo tipo de dispositivos. Además pretende cumplir con los requerimientos establecidos por el workframe 'maintainableCSS'.

Entre las tecnologías utilizadas para la creación de este sitio destacan: html, css, sass, node.js, y server side includes. Se eligieron estas tecnologías por motivos de comodidad y eventual modularización del código fuente.
